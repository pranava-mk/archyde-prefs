# Kitty workspace

## What this is for
Tracks the Kitty terminal emulator user config. HyDE manages `hyde.conf` and `theme.conf` (theme colours, font from theme) — only `kitty.conf` is stowed here.

## Folder contents

| Path inside package | Live symlink target | What it controls |
|---|---|---|
| `.config/kitty/kitty.conf` | `~/.config/kitty/kitty.conf` | Font, font size, padding, tab bar style |

## HyDE-managed files (do NOT stow)
`hyde.conf`, `theme.conf` — regenerated by theme switcher.

## Routing table

| Task | Read | Skip |
|---|---|---|
| Change font or size | `kitty.conf` | `hyde.conf` |
| Change tab style | `kitty.conf` | — |
| Theme colours | HyDE (not this repo) | `kitty.conf` |

## Conventions
- Font: `CaskaydiaCove Nerd Font Mono`, size `12.0`.
- Padding: `window_padding_width 25` (set in `hyde.conf` — override in `kitty.conf` if needed).
- Kitty reloads config live: `Ctrl+Shift+F5` or `kill -SIGUSR1 $(pidof kitty)`.
